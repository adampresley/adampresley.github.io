At first I was greatly intrigued by the fact that you could throw exceptions based off Java classes derived from the *RuntimeException*class. That would look something like this. <cfset illegalArgument = CreateObject("java", "java.lang.IllegalArgumentException").init("Bad argument dude!")> <cftry> <!— Do some code here.
